## Featured Projects

### 🔹 [Trampa Aqui](https://github.com/lucasgfaj/trampa-aqui)
> A job listing system inspired by Thor Vagas, built with **Angular**.  
Focus on responsiveness and usability, tailored for local business opportunities.

### 🔹 [G-Prod](https://github.com/lucasgfaj/g-prod)
> DevOps infrastructure using **Docker** and virtual machine communication.  
Built with **Ruby on Rails** and **MySQL** for a simulated production environment.

### 🔹 [GenBook](https://github.com/lucasgfaj/genbook)
> Full-featured book management system for organizing and cataloging personal or institutional libraries.  
Implements **CRUD operations**, **loan control**, and **pagination** within an **MVC architecture**.  
Built with **Bootstrap** for responsive UI, uses **PostgreSQL** as the database, and runs in a **Docker containerized environment**.
